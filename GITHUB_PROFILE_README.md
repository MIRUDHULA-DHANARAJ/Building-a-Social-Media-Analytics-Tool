## 🚀 Featured Project: Social Media Analytics Engine

**Real-time Sentiment Analysis Platform**

A production-grade analytics tool that collects social media data, analyzes sentiment, and visualizes insights through an interactive dashboard.

### Tech Stack (Simple & Straightforward)
```
Python • Streamlit • VADER • Pandas • Plotly • NLTK • Scikit-learn • PRAW
```

### Key Features

✅ Interactive Streamlit dashboard with live charts  
✅ Reddit API integration via PRAW  
✅ Professional code architecture with logging & validation  
✅ Processes 160K+ social media comments  
✅ Deployable to Streamlit Cloud
