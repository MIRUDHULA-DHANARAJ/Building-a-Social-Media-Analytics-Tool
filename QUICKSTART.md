# Quick Start Guide

## ⚡ Get Started in 5 Minutes

### Option 1: View Sample Dashboard (No setup needed)
```bash
streamlit run app.py
```
Then select "Use Sample Data" - you'll see the full dashboard with our processed Twitter data!

### Option 2: Collect Real Reddit Data (10 minutes)

#### Step 1: Get Reddit API Credentials
1. Go to https://www.reddit.com/prefs/apps
2. Click "Create an App" → Select "script"
3. Fill in the form
4. Copy your credentials

#### Step 2: Run Collection Script
```bash
python collect_reddit_data.py
```
Follow the prompts to enter your Reddit credentials.

#### Step 3: View Dashboard
```bash
streamlit run app.py
```
Upload the processed CSV or refresh to see new data!

### Option 3: Use Your Own Data
```bash
streamlit run app.py
```
Click "Upload CSV" and select any CSV with a `text` column.
